# Power-bank
Basic easy power bank 
it's powered up by two 18650 li-ion batterys 
t6845 battery module 
two wires to change the rack from series to perarlle configuration 
it takes 3.7v as a net voltage from the two batterys
and give 5v from the USB A port as a ( output )
and take 5v from the type c port as ( input )
